## How to Generate PDF

### Easiest Method (Browser)

1. Open `FOLONITE_BUSINESS_OVERVIEW.html` in Chrome, Edge, Firefox, or Safari
2. Press Ctrl+P (Windows/Linux) or Cmd+P (Mac)
3. Select "Save as PDF" as destination
4. Enable "Background graphics" option
5. Click Save

### Automated Method (If Node.js tools available)

```bash
cd /home/engine/project
node convert-to-pdf.js
```

This will attempt to generate PDF using Puppeteer or Chrome command-line.
